SQL Server

SQL Server is a relational database management developed by Microsoft. SQL server supports processing of variety of transactions, analyical processes and business intelligence which helps the company to view and analyse the data. Oracle and My SQL databases are other databases widely used by the companies. There are many versions released in the past years and SQL server 2016 is the latest version. Microsoft says the latest verson has state-of-the-art performance tuning, data analytics and hybrid cloud support.

.Net

.Net is a framwork which is used a development platform. It simplifies the applicaton development by supporting object oriented programming environment. Visual Studio is an integrated development environment (IDE) used to develop .Net applications. Applications can be developed for windows, web and mobile using .Net framework. Visual Studio 2017 is the latest version released by Microsoft.
